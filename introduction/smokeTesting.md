<Smoke Testing> 

O teste de Smoke é o tipo de avaliação que você deve realizar após a implementação de uma nova funcionalidade, pois ele verifica rapidamente se as principais funcionalidades da aplicação estão funcionando corretamente, sem se aprofundar nos detalhes. Isso é exatamente o que você precisa para garantir que a nova implementação não afetou as funcionalidades existentes.

Mínimo de teste para o funcionamento da aplicação.

1. Carga mínima.
2. Cenário Simples.
3. Funcionalidades core.
4. Rápido Resultado.
