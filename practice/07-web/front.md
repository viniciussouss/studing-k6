## Tipos de Testes - Web no contexto de Performace

1. **Navegador**: O teste de carga baseado em navegador verifica o desempenho do frontend de um aplicativo simulando usuários reais usando um navegador para acessar seu site.

2. **Protocolo**: O teste de carga baseado em protocolo verifica o desempenho do backend de um aplicativo simulando as solicitações subjacentes as ações do usuário.

3. **Híbrido**: O teste de carga híbrido é uma combinação de teste de carga baseado em protocolo e baseado em navegador.

### Porque escolher o K6 para testes de Performace no Frontend?
- **Open Source**: O K6 é uma ferramenta de código aberto, o que significa que é gratuita e possui uma comunidade ativa para suporte.
- **Fácil de usar**: O K6 possui uma sintaxe simples baseada em JavaScript, facilitando a criação e manutenção de scripts de teste.
- **Eficiência**: O K6 é leve e eficiente, permitindo simular milhares de usuários com baixo consumo de recursos.
